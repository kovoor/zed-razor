# zed-razor

Zed plugin for `.cshtml` and `.razor` file syntax highlighting.

Adds ASP.NET Razor syntax support to [Zed](https://zed.dev), including Razor directives, inline C# expressions, code blocks, control flow, and HTML with custom/hyphenated tags.

## Install

1. Clone this repo:
   ```
   git clone https://github.com/kovoor/zed-razor.git
   ```

2. In Zed, open the command palette (`Cmd+Shift+P`) and run:
   ```
   extensions: install dev extension
   ```

3. Select the cloned `zed-razor` folder.

4. Reopen any `.cshtml` or `.razor` file to see highlighting.
